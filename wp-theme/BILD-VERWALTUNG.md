# PAXDES Portfolio - Bildverwaltung

## Übersicht

Alle Bilder im PAXDES Portfolio Theme können über das WordPress-Backend verwaltet werden. Es gibt **keine fest codierten Bildpfade** im Theme. Jedes Bild kann über den WordPress Customizer oder über Beitrags-/Seiten-Einstellungen hochgeladen und ausgetauscht werden.

## Zugriff auf Bildeinstellungen

### WordPress Customizer
**Design → Customizer** oder direkt über die Admin-Leiste "Anpassen"

Hier finden Sie folgende Bereiche:

## Wichtige Hinweise

### Bildoptimierung
- Alle Bilder sollten vor dem Upload optimiert werden
- Empfohlene Tools: TinyPNG, ImageOptim, Squoosh
- Zielgröße: < 200KB pro Bild (außer Hero-Bilder: < 500KB)

### Bildformate
- **SVG:** Für Icons und Logos (beste Qualität, kleinste Dateigröße)
- **PNG:** Für Bilder mit Transparenz
- **JPG:** Für Fotos und komplexe Bilder ohne Transparenz

### Responsive Bilder
- WordPress erstellt automatisch verschiedene Bildgrößen
- Das Theme nutzt diese für optimale Performance auf allen Geräten

### Fallback-Bilder
- Alle Bildfelder haben Fallback-Bilder im Theme
- Diese werden angezeigt, wenn kein eigenes Bild hochgeladen wurde
- Fallback-Bilder befinden sich in `/assets/images/`

### Keine Hardcoded Pfade
- **Alle Bilder sind über das Backend verwaltbar**
- **Keine Bildpfade sind fest im Code verankert**
- **Jedes Bild kann jederzeit ausgetauscht werden**

## Schnellzugriff

1. **Logo ändern:** Design → Customizer → Website-Identität → Logo
2. **Hero-Bild ändern:** Design → Customizer → Hero-Bereich → Hero Bild
3. **Signatur ändern:** Design → Customizer → Branding & Logos → Signatur-Bild
4. **Projekt-Bilder:** Projekte → Projekt bearbeiten → Beitragsbild
5. **Hintergrund-Muster:** Design → Customizer → Branding & Logos → Hintergrund-Muster
